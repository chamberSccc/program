algorithm.string:字符类算法
	SumOfWords:给定字符串，返回字符串中出现次数最大的字符。
	LongestPalindrome:寻找字符串最长的回文序列。
	LengthOfLongestSubstring:给定字符串，返回长度最大，且不重复的子串。
	StringToInteger:字符串转换为整形算法。

algorithm.number:数字类算法
	TwoSum:两数相加与目标值一致，返回两数数组索引。
	AddTwoLinkList:两个链表顺序相加，和大于10则进位，返回新链表。
	ReverseNum:给定32位int,返回反转的数字。
	FindMedianSortedArray:两个有序数组，返回中值。

algorithm.utlity:工具
	ArrayToMap:数组转Map
	SimpleLinkList:简单链表实现，未拓展。
